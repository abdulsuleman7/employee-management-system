Employee Management System

This is a Spring Boot based Employee Management System developed to manage employee records efficiently. The application allows users to perform basic CRUD operations such as adding, viewing, updating and deleting employees.

Project Overview

The system is designed to demonstrate backend development using Spring Boot, MVC architecture and database integration. It provides a simple web interface to manage employee details stored in a relational database.

Features

Add new employees
View list of employees
Update employee details
Delete employees
Database integration using JPA and Hibernate


Technologies Used

Java
Spring Boot
Spring Maven
Spring Data JPA
Spring Security
Hibernate
Thymeleaf
MySQL
HTML
CSS
